**GitBoost** is a Chrome extension that enhances GitHub with README table of contents, repository insights panel, keyboard shortcuts, and more.

English | [中文](./README.md)

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📑 **README TOC** | Auto-generates a collapsible table of contents with scroll tracking |
| 📊 **Repo Insights** | View stars, activity, issues/PRs, last commit in sidebar |
| ⌨️ **Quick Nav** | Press `G` for quick navigation menu to Issues, PRs, Wiki, etc. |
| 🌙 **Dark Mode** | Seamlessly adapts to GitHub's dark/light themes |
| 🌐 **Bilingual** | Full Chinese and English UI support |

## 🚀 Installation

### Chrome Web Store
Visit the Chrome Web Store page and click "Add to Chrome".

### Developer Mode
1. Download the source code
2. Open `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the project folder

## 🛠 Development

```bash
git clone https://github.com/your-github-username/gitboost.git
cd gitboost && npm install
```
