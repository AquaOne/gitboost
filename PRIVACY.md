# Privacy Policy

**Last updated:** June 2026

## Data Collection

GitBoost does **NOT** collect any personal data. We do not track your browsing behavior, collect your GitHub account information, or send any data to external servers (except for public API requests to GitHub for the repositories you visit).

## Data Usage

GitBoost only uses the [GitHub Public REST API](https://docs.github.com/rest) to fetch publicly available repository information (such as star counts, issue counts, etc.). This data is identical to what you would see directly on GitHub's website.

All data is processed locally in your browser and is not transmitted to any third party.

## Permissions

- **storage** — Used to save your feature toggle settings
- **host_permissions (github.com)** — Required to inject enhancement features on GitHub pages
- **host_permissions (api.github.com)** — Required to fetch public repository statistics

## Third-Party Services

GitBoost fetches public repository data via the [GitHub API](https://docs.github.com/rest). Use of the GitHub API is subject to the [GitHub API Terms](https://docs.github.com/en/site-policy/github-terms/github-api-terms).

## Updates

This privacy policy may be updated occasionally. Significant changes will be notified through the extension update mechanism.

## Contact

For any questions, please open an issue on the GitHub repository.
