🔒 GitBoost 隐私政策

最后更新：2026 年 6 月

数据收集

GitBoost 不收集任何个人数据。我们不会追踪你的浏览行为，不会收集你的 GitHub 账号信息，不会发送任何数据到外部服务器（除了你浏览的 GitHub 仓库的公开 API 请求）。

数据使用

GitBoost 仅使用 GitHub 公开 REST API 获取仓库的公开信息（如 Star 数量、Issue 计数等），这些数据和你直接在 GitHub 网站上看到的信息完全一致。

所有数据仅在你的浏览器本地处理，不会传输给任何第三方。

权限说明

GitBoost 请求的权限及其用途：

  • storage — 保存你的功能开关设置
  • host_permissions (github.com) — 在 GitHub 页面注入增强功能
  • host_permissions (api.github.com) — 获取仓库公开统计数据

第三方服务

GitBoost 通过 GitHub API 获取仓库公开数据。使用 GitHub API 受 GitHub API Terms 约束。

更新

本隐私政策可能不时更新。重大变更会通过扩展更新通知的方式告知用户。

联系方式

如有任何问题，请在 GitHub 仓库提交 Issue。
