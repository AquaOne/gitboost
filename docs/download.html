📦 安装指南

1
下载扩展文件

点击上方「免费下载」按钮，下载 https://github.com/AquaOne/gitboost/releases/download/v1.0.0/gitboost-v1.0.0.zip 文件到本地 ~70KB

2
解压文件

将 ZIP 文件解压到一个单独的文件夹（如 C:\Extensions\gitboost）

3
打开 Chrome 扩展管理

在 Chrome 地址栏输入 chrome://extensions/ 并回车

4
开启开发者模式

打开页面右上角的「开发者模式」(Developer mode) 开关

5
加载扩展

点击「加载已解压的扩展程序」(Load unpacked)，选择刚才解压的文件夹

6
开始使用！

访问任意 GitHub 仓库页面，就能看到 GitBoost 的功能了！
