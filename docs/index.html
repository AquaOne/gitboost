✨ 核心功能

📑

README 智能目录

长文档自动生成可折叠目录，支持多级标题嵌套。滚动时高亮当前章节，再长的 README 也能轻松导航。

📊

仓库洞察面板

侧边栏一目了然：Stars 趋势、活跃度、Issue/PR 数量、最近提交时间。不用点开每个页面就能了解仓库状态。

⌨️

快捷键导航

按 G 键弹出快速导航菜单，一键跳转到 Issues、PRs、Wiki、Actions 等页面。

🌙

深色模式适配

完美适配 GitHub 的深色/浅色模式，无论你怎么设置，GitBoost 都能无缝融合。

🌐

中英文双语

完整支持中文和英文界面，设置页面、弹窗、提示信息全双语覆盖。

🔒

隐私安全

仅使用 GitHub 公开 API，不收集任何个人数据，不追踪浏览行为，完全开源可审计。

📈

GitHub Trending 中文版

每日更新 Star 增长最快的开源项目，支持按语言和时间范围筛选。

免费在线生成漂亮的 GitHub README 徽章，支持自定义颜色、图标、样式，即改即看。

🚀 安装指南

1
方法一：Chrome Web Store

访问 Chrome Web Store 页面，点击"添加到 Chrome"。
https://chrome.google.com/webstore/detail/...

2
方法二：开发者模式加载

下载源码 → 打开 chrome://extensions → 开启"开发者模式" → 点击"加载已解压的扩展程序" → 选择 GitBoost 文件夹

3
开始使用

访问任意 GitHub 仓库页面，就能看到 GitBoost 的功能了！目录自动出现在 README 右侧，洞察面板在仓库侧边栏。
